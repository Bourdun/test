Введение в bem метатологию(Block Element Modificator)

BEM - Компонетный подход к web разработке. В его основе лежит принцип разделения интерфейса на независимые блоки.
Он позволяет легко и быстро разрабатывать интерфейсы любой сложности и повторно использовать существующий код.
block - функционально независимый компонент страницы.
div class='error' = +
div class='red text' = -
block не должен влиять на другие объекты. Также в css не рекомендуеться использовать bem.

Принцип работы с блоками

Элемент - составная часть блока,
которая не может использоваться в отрыве от него.
Особенности: Название элемента характеризует смысл («что это?» — «пункт»: item, «текст»: text), а не состояние («какой, как выглядит?» — «красный»
: red, «большой»: big). Структура полного имени элемента соответствует схеме: имя-блока__имя-элемента.
Имя элемента отделяется от имени блока двумя подчеркиваниями (__).
Как можно задавать:
div class="block"
div class="block__elem1"
div class="block__elem2"
< class="block__elem3"
/div /div
/div
/div>
Как нельзя задавать имена блокам:
header__container__block
header>container>block
header header_container
header__container__block
/div

Анимация в Css
Создание анимации базируется на ключевых кадрах (key frames),Которые позволяют автоматически воспроизводить
и повторять эффекты на протяжении заданного времени,
а также приостанавливать анимацию по определённому событию

animation-name - определяет список применяемых к элементу анимаций

animation-duration - отвечает за продолжительность анимаций

animation-timing-function - задаёт сценарий развитий анимаций между кадрами

animation-iteration-count - отвечает за повтор проигрывания ключевых кадров

animation-direction - определяет направление и тип проигрывания ключевых кадров

animation-play-state - позволяет запускать или останавливать анимацию по событию

animation-delay - отвечает за задержку перед проигрыванием

animation-fill-mode - определяет какие значения анимированных css свойств применяется к объекту после завершения анимации

Jquery - это библиотека на языке JavaScript. Она подключается к веб-страницам.

Эта библиотека представляет селекторы в стиле CSS,
и методы, которые позволяют собственно находить элементы,
и выполнять с ними какие-то действия.
